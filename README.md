# Handball Cup
Handball Cup is a ***easy-to-use*** application that is used to sets up a ***tournament***.

### Key Features
 - Registrate tournaments
 - Register teams
 - Show all tournaments
 - Autogeneration of match set up
 - Visual presentation of match setup as brackets
